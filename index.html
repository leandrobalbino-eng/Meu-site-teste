CSS é a linguagem que usamos para estilizar um documento HTML. CSS descreve como os elementos HTML devem ser exibidos. Este tutorial lhe ensinará CSS do básico ao avançado.

Tipos de CSS

  • Inline- usando o styleatributo dentro de elementos HTML
  • Interno- usando um style elemento na head seção
  • Externo - usando um link elemento para vincular a um arquivo CSS externo
